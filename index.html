Who's Behind It

About Villitech

Villitech Dezines & Graphix is a Fresno, California design and IT consulting studio run by Joseph Villicano. The work spans two disciplines that don't usually share a desk: visual design and hands-on technical support.

That combination means projects get built the way they're meant to run, not just look good in a mockup. A logo gets handed off in the right file formats. A website gets built to actually load fast and stay online. A flyer gets sized correctly for the printer before it's ever sent.

Every project starts with the same question: what does this actually need to do its job well? The rest, the gears and the brass trim, is just how that gets delivered.
